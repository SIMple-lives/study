# C++ explicit

**explicit作用**:

在c++中，explicit关键字用来修饰类的构造函数，被修饰的构造函数的类，不能发生相应的隐式类型转换，只能以显示的方式进行类型转换.

**注意事项**

* elplicit关键字只能用于类内部的构造函数声明上
* explicit关键字作用于单个参数的构造函数
